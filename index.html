Kebijakan Privasi

StokGudang adalah aplikasi manajemen stok gudang yang dibuat oleh ailuasc Creative.

Data Anda 100% privat. Semua data (produk, opname, riwayat) tersimpan hanya di localStorage browser perangkat Anda sendiri. Tidak ada data yang dikirim ke server manapun.

Kami tidak mengumpulkan, menjual, atau membagikan data apapun. Tidak ada akun, tidak ada cloud, tidak ada tracking.

Transfer Data menggunakan kode yang Anda generate dan salin sendiri — kami tidak menyimpan kode tersebut.
